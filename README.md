# Axos Bank Finance Portal

## Phase 1: Static HTML Prototype

This is the initial Phase 1 development of the Axos Bank Finance Portal, a web-based platform for managing financial deals, documents, and communications with underwriters.

### Project Structure

```
.
├── index.html              # Main dashboard page
├── css/
│   ├── styles.css         # Main styles
│   ├── responsive.css     # Mobile responsive styles
│   └── auth.css           # Authentication page styles
├── js/
│   ├── main.js            # Main JavaScript
│   ├── auth.js            # Authentication logic
│   ├── deals.js           # Deals management
│   ├── documents.js       # Document management
│   ├── messages.js        # Messaging functionality
│   └── settings.js        # User settings
└── pages/
    ├── login.html         # Login page
    ├── deals.html         # Deals management page
    ├── documents.html     # Document management page
    ├── messages.html      # Messaging page
    └── settings.html      # User settings page
```

### Features Implemented

- ✅ Static HTML structure for all core pages
- ✅ Responsive CSS design (mobile, tablet, desktop)
- ✅ Navigation menu
- ✅ Dashboard with quick stats
- ✅ Deals overview and management
- ✅ Document upload interface
- ✅ Messaging interface
- ✅ User settings and profile management
- ✅ Login/authentication pages
- ✅ Accessibility features (ARIA labels, semantic HTML)

### Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Navigate through the pages using the menu

### Development Notes

- All HTML is semantic and includes ARIA labels for accessibility
- CSS uses a mobile-first responsive design approach
- JavaScript is organized by feature module
- All code follows consistent naming conventions
